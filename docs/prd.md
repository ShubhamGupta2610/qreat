# RQEAT Online Ordering System Requirements Document

## 1. Website Overview

### 1.1 Website Name\nRQEAT\n
### 1.2 Website Description
An online restaurant ordering platform that enables customers to browse menus, place orders, track spending, and receive personalized discounts. The system includes an AI-powered feedback agent to assist customers throughout their ordering experience.

## 2. Core Features

### 2.1 User Authentication
- Login page for existing users
- Signup page for new user registration
- Users must login before placing orders
- Collect and save customer details during registration:
  - Name
  - Mobile number\n  - Table number
\n### 2.2 Customer Profile Management
- Store customer information: name, mobile number, table number
- Display customer details in user profile
- Show total amount spent by customer in profile section
- Allow customers to update their information

### 2.3 Menu Browsing
- Display menu items organized by categories: Starter, Main Course, Dessert, Fast Food
- Show item details including name, description, price, and images
- Search and filter functionality for easy navigation

### 2.4 Order Management\n- Add items to cart with quantity selection
- Review cart contents before checkout
- Place orders with order confirmation
- Associate orders with customer details (name, mobile number, table number)
- Order status tracking (Received, Preparing, Delivered)

### 2.5 Payment
- Integrated payment system for order checkout
- Support multiple payment methods
- Payment confirmation and receipt generation
- Display total amount spent after payment completion

### 2.6 Customer Analytics
- Track and display total amount spent by each customer\n- Show spending history in customer profile
- Count total number of customers
- Automatic discount calculation based on spending history\n
### 2.7 AI Feedback Agent
- Interactive AI assistant to help customers with:\n  - Menu recommendations
  - Order assistance
  - Answering questions
  - Collecting feedback

### 2.8 Discount System\n- Automatically provide discounts based on customer spending
- Display discount information clearly during checkout

### 2.9 Admin Dashboard
- Admin login with secure authentication
- Menu Management:
  - Add new menu items
  - Edit existing items
  - Delete items
  - Update prices and availability
- Discount Management:
  - Set discount rules and percentages
  - Manage promotional offers
- Order Management:\n  - View incoming orders with customer details (name, mobile number, table number)
  - Update order status (Received → Preparing → Delivered)
  - Track order history
- Customer Management:
  - View customer list with details
  - Display total spending amount for each customer
  - Track customer spending and visit count\n
## 3. Design Style

### 3.1 Visual Theme
- Warm and cozy atmosphere with earthy tones: soft browns (#8B7355), cream (#F5E6D3), warm orange accents (#E67E22)
- Soft lighting effects and gentle shadows to create inviting ambiance

### 3.2 Typography & Layout
- Rounded, friendly fonts for headings with comfortable reading size
- Card-based layout with subtle rounded corners (8-12px radius) for menu items\n- Generous spacing between elements for relaxed browsing experience
\n### 3.3 Interactive Elements
- Smooth hover transitions with gentle scale effects
- Warm glow effects on buttons and interactive components
- Cozy icons with hand-drawn or soft-edge style

### 3.4 Overall Composition
- Clean, uncluttered interface with focus on food imagery
- Warm background textures (subtle wood grain or fabric patterns)
- Comfortable visual hierarchy that guides users naturally through the ordering process
